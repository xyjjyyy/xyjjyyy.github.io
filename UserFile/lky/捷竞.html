捷竞

文件结构

正如您所见，以下三个文件夹是我们网页部分全部的源码，包含官方主页(www.sxl.cn)和注册登录以及论坛界面(JieJing)以及后端服务(jjServe);

工具使用

我们主要使用代码编写工具、数据库存储工具、域名映射工具以及线上部署工具。

  • 使用了 webStrom (或者 vscode )进行代码部分的编写；
  • 借助 mysql 作为本地数据库进行登录信息、登陆验证以及论坛信息的储存；
  • 采用 花生壳 进行 http https 公网域名映射(以减少域名方面的注册花费)；
  • 利用 腾讯云 CloudBase 进行云上部署；

如何运行

本部分讲解如何运行全部的代码

我们进入 互联网加创业实训网页源码这个文件夹进行操作

因为数据库Mysql部署在本地及笔者的电脑上，原则上来说你是无法在你的电脑上运行这些文件的，但是笔者假设你会修改数据库部分代码，并且笔者侧重于讲解我们运行的方法以及如何部署。

提前npm install -g @cloudbase/cli安装好我们的 cloudbase/cli 工具

  • Mysql 建表包括 logtable 和 luntan (从上至下)

  • 官方主页(www.sxl.cn)线上部署：

    • 进入文件夹 cd www.sxl.cn
    • cloudbase初始化 cloudbase init --without-template
    • 部署 cloudbase framework:deploy

  • 注册登录以及论坛界面(JieJing)线上部署：

    • 进入文件夹 cd JieJing
    • cloudbase初始化 cloudbase init --without-template
    • 部署 cloudbase framework:deploy

  • 后端服务(jjServe)部署：

    • 花生壳域名映射服务打开(免费的两个域名映射就够用了)
    • 进入文件夹 cd jjServe
    • 运行后端 node app.js

然后环境就全部配置完成了，云上部署也就全部完成了，由于后端部署需要租用云服务器才行，为了减少花费决定采用本机作为服务器，所以导致了登陆界面只有在本机电脑进行打开的情况下才可以进行访问,所以如有需要可以联系 笔者 (QQ:904588748) 进行实践。
